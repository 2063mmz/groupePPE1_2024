D'abord, Vérifie si l'utilisateur a bien fourni deux arguments. if [[ $# -ne 2 ]] Définition des variables globales dossier=$1, base=$2 Et définir les fichiers d'entrée et les fichiers de sortie:

Ensuite, Traitement des textes par langue. Pour l'Espagnol, les fonctionnalités sont : Ignore les lignes vides. Extrait les mots et la ponctuation grâce à une expression régulière. Écrit le texte formaté dans le fichier contextes. Pour l'Anglais: Pour dumps-text Sépare les mots et la ponctuation avec sed et tr. Supprime les lignes vides. Écrit le texte formaté dans le fichier dumps-text. Pour contextes Extrait les mots et ponctuations. Écrit le résultat dans le fichier contextes.

Pour le Chinois, Extrait chaque caractère chinois et ponctuation. Écrit le résultat dans le fichier dumps-text.

A la fin, Les fichiers de sortie sont enregistrés dans le dossier ../pals/, avec les noms: dumps-text-<base>.txt et contextes-<base>.txt.
